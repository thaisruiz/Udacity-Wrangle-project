Act Report¶

In this project, the dataset to analyze and visualize is the tweet archive of Twitter user @dog_rates, also known as WeRateDogs. This Twitter account rates peoples dogs with a humorous comment about the dog. It was started in 2015.

The data contains basic tweet data for all 5000+ of their tweets. A provided enhanced version filtered for tweets with ratings only (2356 records). However, this version needed further assessment and cleaning to be able to use it for analysis and visualization.

After the data wrangling process, the Twitter Archive master file ended up containing 1968 records. Not all of the detected (tidiness and quality) issues were addressed per the scope of this project since it would require a lot of time; but this final version provided with a file suitable for performing EDA.

Out of 297 dogs (which are the categorized ones), 68% are classified as puppers, 21.5% as doggo, 7.7% as puppo and 2.7% as floofer. These proportions might be the trend, however, the uncategorized dogs need to be filled out with their corresponding stage to show or not if these numbers are kept.

Althought the data wrangling is time consuming, it is neccesary to invest time on assessing and cleaning the most it can be, in order to be able to analyze and visualize a more accurate data and get real results. The analysis without cleaning or over a still dirty-messy data might give, pretty much, wrong insights and therefore a wrong perception of the data. Also, it might take more time time and complexity to handle it.

The current analysis and visualizations are based on an assessed and clean data, but we need to keep in mind there is still more room for further wrangling work.
